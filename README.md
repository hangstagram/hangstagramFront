### 코드컨벤션

```
- Feat : 기능 구현
- body : 세부 내용
- Fix : 버그 수정
- conflict : 깃 push 오류 수정
- CSS : 페이지 스타일링
- Refactor : 코드 리펙토링 작업
- Folder : 폴더 구조 작업
- etc : 기타작업(환경변수 설정 등)
```

### 담당기능
```
모세: 업로드 페이지. 로그인페이지
- 게시글 업로드
- 이미지 업로드
- 로그인 기능
- 헤더 컴포넌트 작업
- 프로젝트 배포

종화: 메인 페이지, 회원가입 페이지
- 데이터 조회
- 삭제 기능
- 게시글 클릭시 상세페이지
- 회원가입 기능
```
